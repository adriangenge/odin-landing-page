Software How-to Guide

Project Description

This how-to guide was written to help the assistants on my team update our website. It contains step-by-step instructions, images, helpful links, and clear explanations of the tasks needed to keep the website updated. These tasks include setting up secure accounts, customizing the appearance of the website while meeting required design constraints, adding content to the website, and accessing Google Analytics reports to see website traffic. The how-to guide was written for a beginner audience with no experience in WordPress or web design, and therefore contains simple language and easy to follow steps.
