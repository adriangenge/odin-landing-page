WordPress Website

Project Description

This website was created to provide stakeholders with information and updates on the research conducted by the small research team of which I was a part of. The design of the website was carefully crafted to effectively communicate the teams progress while complying with the standards set by the overarching educational institution. The website was created on WordPress using a modified version of the UBC Collab WordPress Theme. Search engine optimization best practices and Google Analytics were used to increase and monitor website traffic.

Skills + Tools Used

  • WordPress
  • HTML/CSS
  • Search engine optimization
  • Google analytics
  • Information architecture
